# v1.1.0 发布说明

> **结构优化与兼容性增强** - 2025年12月

## 🏗️ 项目结构重构

为了提升代码组织的整洁度，我们对项目目录结构进行了深度重构：
- **入口更名**：原 `toolbox.py` 更名为 `main.py`，作为统一的程序入口。
- **脚本分类**：
  - PowerShell 脚本移至 `scripts/ps1/` 目录。
  - Python 辅助脚本（如 `remove.py`）移至 `scripts/python/` 目录。
- **智能扫描**：更新了脚本扫描逻辑，现在支持递归扫描 `scripts/` 下的一级子目录，方便未来按类别组织更多脚本。

## 💻 Windows Terminal 安装增强 (重点)

针对 LTSC 和 Server 系统的安装体验进行了大幅改进：
- **依赖自动处理**：新增 `Microsoft.UI.Xaml.2.8` 框架的自动检测与安装。脚本会自动从 NuGet 获取缺失的依赖，解决 HRESULT: 0x80073CF3 错误。
- **全系列适配**：增强了对 Windows Server 2019/2022/2025 和 LTSC 2019/2021/2024 的 Build 号识别逻辑，自动选择最佳安装包。
- **交互优化**：安装完成后新增手动冷重启（注销/重新登录）提示，并支持在独立运行时一键注销，确保 `wt` 命令立即生效于 PATH。

## 🛠️ 功能优化

- **系统识别修复**：修复了在某些 Server 版本中系统类型识别重复显示的 Bug。
- **文档体系完善**：
  - 新增 `docs/utf8-support.md` (UTF-8 支持工具说明)。
  - 新增 `docs/security-notifications.md` (安全中心通知禁用说明)。
  - 深度更新 `docs/windows-terminal.md`，涵盖新版依赖处理逻辑。
- **README 更新**：同步了最新的目录结构、启动命令及系统支持表格。

## 🐛 严重 BUG 修复

- 修复了打包版无法运行的BUG。现在项目只专注于从源码运行。

## 🔒 许可证与规范

- 优化了项目中的注释和元数据解析逻辑。
