# v1.0.0 发布说明

> **首个正式版本** - 2025年12月

## 🎨 图形化界面

- **全新的 Fluent Design 界面**：基于 PySide6 和 QFluentWidgets 开发，提供现代化的 Windows 11 风格 UI
- **深色主题**：终端窗口采用 Catppuccin Mocha 配色，护眼且美观
- **平滑滚动**：工具列表支持流畅的滚动体验
- **自动居中显示**：窗口启动时自动居中于屏幕

## 🛠️ 核心功能

### 工具管理
- **脚本元数据解析**：自动从脚本注释中提取标题和描述，无需手动配置
- **工具卡片**：每个工具以卡片形式展示，包含图标、标题、描述和运行按钮
- **并发任务执行**：支持同时运行多个工具任务，互不干扰

### 任务界面
- **实时日志输出**：终端风格的日志显示，支持自动着色
- **UTF-8/GBK 编码自动处理**：智能检测并正确显示脚本输出
- **复制日志**：一键复制完整日志到剪贴板
- **状态指示**：任务完成后显示成功/失败状态

### 缓存管理
- **智能缓存**：下载文件自动缓存到 `tmp` 目录，避免重复下载
- **自动清理**：程序正常退出时自动清理临时文件
- **环境变量传递**：通过 `TOOLBOX_TMP_DIR` 环境变量与脚本共享缓存路径

## 📦 内置工具

1. **PowerShell 7 安装工具** (`Install-PowerShell7.ps1`)
   - 自动检测当前版本，判断是否需要更新
   - 从 GitHub API 获取最新版本信息
   - 支持 x64 和 ARM64 架构

2. **Microsoft Store 安装工具** (`Install-MicrosoftStore.ps1`)
   - 为 LTSC/Server 系统安装 Microsoft Store
   - 自动下载所需的 Appx 依赖包

3. **Windows Terminal 安装工具** (`Install-WindowsTerminal.ps1`)
   - 安装最新版 Windows Terminal
   - 自动处理依赖项

4. **安全中心通知禁用工具** (`Disable-SecurityNotifications.ps1`)
   - 通过注册表禁用 Windows 安全中心的弹出通知

## 🧪 开发与测试

- **100% 测试覆盖率**：49 个单元测试用例，覆盖所有核心功能
- **自动化 CI/CD**：
  - CI 工作流：每次提交自动运行测试和构建验证
  - Release 工作流：手动触发，自动打包并发布到 GitHub Releases
- **Nuitka 打包**：发布版使用 Nuitka 编译为高性能单文件 exe

- 管理员权限
